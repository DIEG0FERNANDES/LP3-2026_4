# Prática de Express

Este projeto foi desenvolvido como prática da disciplina **Linguagem de Programação III**, ministrada pelo professor **Leandro Guedes**.

## Objetivo

Explorar conceitos básicos de desenvolvimento backend utilizando o framework **Express.js** no ambiente **Node.js**.

## Conteúdo

- Configuração inicial do servidor Express
- Criação de rotas
- Manipulação de requisições e respostas
- Estruturação de um projeto simples

## Observações

Este repositório é apenas para fins acadêmicos e de aprendizado.
